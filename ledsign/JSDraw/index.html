LED Sign

Use this page to draw something on my LED sign. This page uses JavaScript. You can toggle a LED by clicking on it. To upload the image to the LED sign click submit. You can see a log file of previous pictures (make sure you refresh to get the latest copy). You may be able to connect to a webcam. If that doesn't work you can get a single image (again, make sure you refresh). If neither of these work the webcam software is probably not running. You can also print text instead.

If this doesn't work for you try the text only drawing program.
